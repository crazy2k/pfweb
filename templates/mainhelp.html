Información general

Este es el "planifi web", un sistema para los estudiantes de la Facultad de Ingeniería de la UBA, que te ayuda en la planificación de tu carrera.

La mejor forma de conocer bien de que te sirve es probarlo, asi que si todavía no tenes un usuario, te recomendamos crearte uno y ver que te parece.

¿Es un sistema oficial de la facultad?

Este sistema NO es un sistema oficial de la facultad.
No tiene ninguna relacion con el cuerpo docente ni no-docente de la facultad, ni con el CEI ni ninguna agrupación política estudiantil.
Fue creado y es mantenido por alumnos de la facultad, podes ver abajo mas detalles.

Los datos que agregues aca no tendran ninguna repercusión sobre tus datos administrativos oficiales, ni sobre el sistema de inscripciones.

¿Quien lo hizo?

La idea y el sistema original fueron desarrollados por Diego Essaya. Luego varios de los chicos y chicas del LugFI colaboraron de distintas maneras para la creación de esta interfaz web.

¿A quien le aviso si algo no anda, o anda mal?

Para cualquier tipo de comentarios, reportes de problemas, sugerencias u otras consultas, manda un mail a Alberto Bertogli (albertito@gmail.com).

Información técnica

El sistema se compone de dos partes: un server XML-RPC que hace de backend (llamado planifid), y que es quien implementa toda la lógica, persistencia de datos y manejo de sesiones; y un frontend web a modo de interfaz (llamado pfweb).

Ambas partes estan escritas usando el lenguaje de programación Python; para el desarrollo del backend no se utilizaron bibliotecas externas, y para el frontend web se usó el framework web.py.
